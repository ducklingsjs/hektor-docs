# styles

## Status: Partially implemented

## Subtasks

  * sass
  * autoprefixer

## Options

  * src
    * entry point
    * default: {app}/styles/main.scss
  * includePaths
    * paths that are automatically included
    * default: {app}/bower_components
  * browsers
    * browser support needed
    * default: ['last 2 versions', 'ie 10']
  * dest
    * destination path
    * default: .tmp/styles
